# Math Util

## Notes
* Adds formulas to calculate distance, and other stuff
* No need to write it from scratch

## Usage
```cs
// Get distance with floating point precision
double dist = MathUtil.GetDistance2D(x, y, x2, y2);

// Same as above, but converts into int
int dist = MathUtil.GetDistance2DInt(x, y, x2, y2);
```
